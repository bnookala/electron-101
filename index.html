Electron allows you to create cross platform desktop applications using plain ol' Javascript!


It's (probably) in your life already


Atom, Nylas, Wordpress, and many more…

Getting started

npm install -g electron-prebuilt
or…
npm install --save electron-prebuilt
If you've got an existing node package, you can use --save

Hello Electron

hello-electron.js
So we've created a super simple electron app. It does literally nothing, except print to a terminal. But the electron main process itself stays active, waiting for events from a different process.

Two different processes

Source
Electron wraps two processes, a 'main' process, and a 'renderer' process. They both communicate to each other via IPC, and both have full access to the Node.js and Electron APIs.
